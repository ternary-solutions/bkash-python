# Support

Use the right channel so we can help quickly and keep the issue tracker focused.

## GitHub Discussions

Use
[Discussions](https://github.com/ternary-solutions/bkash-pgw-tokenized-python-package/discussions)
for:

- Integration questions
- Usage examples
- API design ideas
- General SDK support

## GitHub Issues

Use [Issues](https://github.com/ternary-solutions/bkash-pgw-tokenized-python-package/issues)
for:

- Reproducible bugs
- Documentation problems
- Concrete feature requests
- Regressions between releases

## Security

Send private vulnerability reports to `devsecops@ternary.solutions`.

## Upstream bKash Platform Questions

If the problem appears to be with merchant onboarding, account provisioning, or
the upstream bKash platform rather than this library, refer to the official
bKash developer documentation and support channels:

- https://developer.bka.sh/

When opening a public support request, use sandbox data and redact all secrets.
