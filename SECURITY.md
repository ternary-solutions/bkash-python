# Security Policy

The `bkash-pgw-tokenized` maintainers take security reports seriously. Please
report suspected vulnerabilities privately so we can investigate and coordinate
a fix before public disclosure.

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.x     | Yes       |
| < 1.0   | No        |

## Reporting A Vulnerability

Email `devsecops@ternary.solutions` with:

- A short description of the issue
- Impact and affected versions
- Reproduction steps or a proof of concept
- Any suggested mitigations if you have them

Please do not open public GitHub issues for security vulnerabilities.

## Response Expectations

- Initial acknowledgment within 3 business days
- Ongoing status updates while we investigate
- Coordinated disclosure after a fix or mitigation is available

If your report includes logs, requests, or payloads, redact credentials,
personal data, and live transaction details before sending them.
